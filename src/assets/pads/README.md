# Sound pad icons

Place pad icon images here. The app also checks `public/pads/` at runtime.

## Required files

Same names as characters (one icon per pad):

- `beat-1.png` … `beat-5.png`
- `melody-1.png` … `melody-5.png`
- `effect-1.png` … `effect-4.png`
- `percussion-1.png` … `percussion-5.png`
- `voice-1.png` … `voice-5.png`

Recommended: square, ~128×128px or larger, simple bold icon on transparent background (colour comes from the pad tile CSS).

If a file is missing, the app shows a simple SVG icon fallback.
